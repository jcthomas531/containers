# Overview

This tutorial covers:

* how to set up a wsl environment to work with containers
* how to make the build files for a container
* how to build a container
* how to transfer that container out of wsl and into a folder on windows and then to the hpc



# WSL environment for containers

The first step is to actually have WSL for on your machine. This has to be installed by IT. The second step is having some sort of thing like Ubuntu so that you can interact with WSL. Ubuntu can be downloaded via the Microsoft store without any help from IT.
